Политика конфиденциальности

Ultimate AI — ИИ‑ассистент (далее — «Бот») соблюдает конфиденциальность пользователей и обрабатывает персональные данные в соответствии с Федеральным законом № 152‑ФЗ «О персональных данных».

Какие данные собираются

  • Telegram ID — для идентификации пользователя и предоставления доступа к оплаченным пакетам запросов.
  • Имя пользователя (username) — для персонализации общения.
  • История диалогов с Ботом — для поддержания контекста беседы (хранится в течение всего периода использования Бота).

Цели обработки

Данные используются исключительно для:

  • оказания услуги (обработка запросов, ведение истории);
  • формирования фискальных чеков (в соответствии с 54‑ФЗ);
  • улучшения качества сервиса.

Передача третьим лицам

Бот не передаёт персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ (например, по запросу уполномоченных органов).

Хранение и удаление

Данные хранятся в течение всего периода использования Бота. История диалогов может быть удалена по запросу пользователя, направленному в поддержку @ultimate_bot_support. При удалении данных неиспользованные оплаченные запросы аннулируются без компенсации.

Согласие

Используя Бота, вы соглашаетесь с данной политикой. Если вы не согласны, пожалуйста, прекратите использование Бота.
